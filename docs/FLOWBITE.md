> Flowbite is an open-source library of over 600+ UI components, sections, and pages built with the utility classes from Tailwind CSS and designed in Figma. It features a robust JavaScript API for interactive elements and supports multiple frameworks including React, Vue, Svelte, Angular, Next.js, and more. Open Source. Tailwind CSS-based. Figma-Ready.
