# Categorias de Productos (Tienda)

Este documento define la lista oficial de categorias y subcategorias para la pagina de inicio de la tienda.

## 1. Salas
- Sofas
- Sillones individuales
- Sofas de dos plazas

## 2. Comedores
- Mesas de comedor
- Sillas de comedor
- Bancos

## 3. Recamaras
- Camas
- Cabeceras
- Buros

## 4. Closets y almacenamiento
- Closets
- Roperos
- Armarios

## 5. Escritorios y oficina
- Escritorios
- Sillas de oficina
- Estaciones de trabajo

## 6. Muebles para TV
- Centros de entretenimiento
- Bases para TV
- Consolas

## 7. Mesas
- Mesas de centro
- Mesas laterales
- Mesas auxiliares

## 8. Estanterias y libreros
- Libreros
- Repisas
- Estantes

## 9. Cocina
- Alacenas
- Islas de cocina
- Gabinetes

## 10. Muebles infantiles
- Camas infantiles
- Escritorios para ninos
- Organizadores

## 11. Muebles decorativos
- Consolas decorativas
- Biombos
- Bancos decorativos

## 12. Muebles personalizados
- Disenos a medida
- Proyectos especiales
- Muebles bajo pedido

## 13. Muebles de jardin
- Salas de exterior
- Comedores de exterior
- Camastros y tumbonas

## Notas
- Esta lista se usa como referencia para mantener sincronizada la pagina de inicio.
- Si se modifica una categoria aqui, se debe reflejar en el componente [app/templates/components/ecommerce/categories.html](../app/templates/components/ecommerce/categories.html).
- Para el dataset inicial completo (productos, colores, tipos de madera, BOM y costos), consulta [INITIAL_SEED_BLUEPRINT.md](./INITIAL_SEED_BLUEPRINT.md).
